# Sentinel Orchestrator v5

高階經理人智囊團 — 戰略決策支援系統

## 特色

### 👔 高階經理人視角
- **Gemini** — 戰略長（CSO）：戰略思維、利益相關者分析、組織能力評估
- **NVIDIA** — 營運長（COO）：執行力、組織擴展、流程優化
- **Perplexity** — 策略情報長：市場趨勢、競爭分析、弱訊號捕捉
- **Grok** — 創業家：快速行動、大膽決策、質疑穩健
- **OpenAI** — 風控長：長期主義、系統性風險、永續發展
- **Vertex** — 董事會主席：綜合視角、利益平衡、辯證思維
- **Local** — 獨立顧問：獨立思考、不受干擾、回歸本質

### 📱 完整 iPhone 適配
- 自動隱藏側邊欄（手機版）
- 響應式字體與間距
- 觸控優化按鈕
- 流暢滑動體驗

### 🎯 四種決策模式
1. **快速模式** — 戰略長 + 營運長快速診斷
2. **預測模式** — 加入情報長的市場洞察
3. **深度研究** — 全智囊團綜合分析
4. **AI 辯論** — 正方(創業家+風控) vs 反方(營運+戰略) / 主席裁決

## 部署

已部署至 Vercel：https://sentinel-orchestrator.vercel.app

## 環境變數

在 Vercel 設定以下 API Keys（設定一次，所有裝置通用）：

```
NEXT_PUBLIC_GEMINI_KEY
NEXT_PUBLIC_GROK_KEY
NEXT_PUBLIC_PERPLEXITY_KEY
NEXT_PUBLIC_OPENAI_KEY
NEXT_PUBLIC_NVIDIA_KEY
NEXT_PUBLIC_VERTEX_KEY
```

## 本地開發

```bash
npm install
npm run dev
```
